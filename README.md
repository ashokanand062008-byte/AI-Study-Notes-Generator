# 🚀 AI Study Notes Generator

An AI-powered study notes generator built using HTML, CSS, JavaScript, Flask, and AI APIs.

Generate clean, structured notes instantly with a futuristic neon-inspired interface.

## ⚙️ Installation

Clone repository:

```bash
git clone YOUR_REPOSITORY_LINK
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Flask:

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
